# 零、构建说明
按以下步骤构建一个页面
1. 构建之前，通读本文全文，及本项目内所有的指导文档，包括但不限于知识库文件，AI-PIPELINE设计、买家匹配AGENT设计、前后端方案；如果前后端方案与本文有冲突，先列出来告知我，然后再开始做
2. 构建时，需要按以下步骤进行：  
（1）读对应地址的图片，并根据描述，实现页面，注意做的时候要完全复刻图片，并完全实现下方逻辑，代码内需要做好注释  
（2）若图片与描述有冲突，以描述为准，先实现，并做好较为统一的代码注释  
（3）涉及到其他图/页面的部分以对应页面的要求进行设计与跳转，重点关注跳转目的页，这个思路应该贯穿所有页面设计  
（4）你的设计需要使用你的所有前端skills，重点是impeccable这个skill，我知道前端方案里已经有所规定，但那只是颜色，你整体来说还是使用前端skills保证实现效果与原型图几乎一致  
（5）最终目的：产出页面之间跳转流程符合逻辑与描述的完整页面  
（6）不要直接写HTML，用框架，页面完全按H5（适配手机）的尺寸进行构建，并且每一页的所有内容要能在一屏内展示完
